# eTicket Backend

eTicket is a WebPlatform that provides you easy fast and secure payments of your BRIDGE, TOLL, PARKING and etc taxes

## Installation

You will need gradlew and docker for this

## Usage

```bash
cd local
docker-compose -f mysql-docker-compose.yml up
cd ..
./gradlew build
```
